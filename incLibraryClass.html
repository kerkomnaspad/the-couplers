Bad Code Smell : The Couplers

Incomplete Library Class

Class yang memiliki fungsionalitas, tetapi tidak di-implementasi dengan benar.


Contoh code:

public class PaymentProcessor {
  // Metode untuk memproses transaksi kartu kredit
  public void processCreditCardTransaction(String cardNumber, double amount) {
   // kode untuk memproses transaksi kartu kredit
  }
}


Sebagai program yang memproses transaksi, hanya memproses kartu kredit tidaklah cukup. Maka sebaiknya method ditambah.



Setelah refactor:

public class PaymentProcessor {
  // Metode untuk memproses transaksi kartu kredit
  public void processCreditCardTransaction(String cardNumber, double amount) {
   // kode untuk memproses transaksi kartu kredit
  }

  // Metode untuk memproses transaksi melalui PayPal
  public void processPayPalTransaction(String email, double amount) {
   // kode untuk memproses transaksi melalui PayPal
  }

  // Metode untuk memproses transaksi cryptocurrency
  public void processCryptoTransaction(String walletAddress, double amount) {
   // kode untuk memproses transaksi cryptocurrency
  }
}

Solusi: Menambahkan lebih banyak method yang dapat digunakan dalam transaksi.
